DreamWare
=========

☆마피아를 잡아라

대학생의 낭만 MT 꽃 마피아 게임
어플리케이션으로 부활하다.


사용자에게 주는 가치
1. 마피아 게임은 여러 사람이 모여서 하는 게임이지만 쉽게 상대 유저와 만나서 쉽게 즐길 수 있다.
2. 간단한 룰로 인한 남녀노소 쉽게 즐길 수 있다.
3. 오프라인상에서 게임 중 불편 함 해소
(죽은사람이 힌트를 준다, 움직이는 소리 등으로 눈치를 챈다)

주요기능
1. 실시간 유저간의 채팅
2. LEVEL과 승률 시스템  
3. 게임 중 나간 유저 페널티 시스템

사용언어 :java
개발문야 :안드로이드 어플리케이션
